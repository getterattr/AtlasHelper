# Glossary

Canonical domain vocabulary for AtlasHelper. Devoid of implementation detail.

## Phase
One of the four ordered stages of league-start atlas progression:
1. **Phase 1 - First Voidstone**: reach T16 in the bottom-left corner and defeat Exarch + Eater for voidstone 1.
2. **Phase 2 - Ten-Way Maven**: complete the Maven's 10-map invitation and place voidstone 2 (top-left corner, Originator).
3. **Phase 3 - Full Completion**: reach 100% atlas bonus completion across all non-unique maps.
4. **Phase 4 - Final Voidstones**: self-farm voidstones 3 and 4 (Decayed via Shaper + Elder in the top-right corner; Ceremonial via the Maven in the bottom-right corner).

## Bonus Completion
The one-time "+1 atlas passive point" awarded per map for killing its boss at or above the map's native tier and at or above the rarity printed on the atlas node: **magic** for whites (T1-T5), **rare** for yellows (T6-T10), **rare + corrupted** for reds (T11-T16).

## Corner
One of the four extreme regions of the atlas, each gated by a pinnacle encounter and each accepting one Voidstone.

## Voidstone
An atlas-level upgrade item socketed into a Corner. Each socketed Voidstone raises all map tiers by four. Four total.

## Quest Chain
An ordered sequence of tier-gated map runs required to spawn a pinnacle boss invitation. Two run in parallel from white maps onward: the **Exarch chain** and the **Eater chain**.

## Eagon
NPC Eagon Caeserius. Spawns a memory tear on the player's first T14+ map when no Eagon chain is already active, and respawns on the next T14+ after the previous chain finishes. Each chain hands the player the three memory maps that make up the *Threads of the Originator* questline.

## Envoy
NPC who introduces the Exarch and Eater questlines in a yellow (T6+) map, handing over the **Luminous Astrolabe** and **Flesh Compass** used to add Exarch and Eater influence at the Map Device.

## Maven's Beacon
A Map Device toggle awarded after first meeting the Envoy. Enabling it invites the Maven to witness the map boss; killing that boss registers a witness toward the Maven's Invitation ladder.

## Run Priority
The rule set that picks the next map to run. Default rule: run the highest-tier map available. Exception: when a Quest Chain requires a specific tier to progress, run that tier instead.
